WATER JUG PROBLEM - PSEUDO CODE

BEGIN

1. Define the capacities of Jug 1 and Jug 2.
2. Define the target amount of water.
3. Create an empty queue and an empty visited set.
4. Insert the initial state (0, 0) into the queue.

5. WHILE the queue is not empty DO
      a. Remove the first state from the queue.
      b. IF the state is already visited THEN
            Continue to the next state.
      c. Mark the current state as visited.
      d. IF the target amount is present in either jug THEN
            Display the sequence of states.
            STOP.
      e. Generate all possible next states:
            - Fill Jug 1
            - Fill Jug 2
            - Empty Jug 1
            - Empty Jug 2
            - Pour water from Jug 1 to Jug 2
            - Pour water from Jug 2 to Jug 1
      f. Add all unvisited states to the queue.
   END WHILE

6. Display "No solution exists" if the target cannot be reached.

END
